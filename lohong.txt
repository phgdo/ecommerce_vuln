- login.php:
    SQL Injection qua $username (no escape, no prepared statement).
    Plaintext password so sánh trực tiếp, lưu cookie dạng id:username:password.
    Session fixation cho phép set sid qua GET.
    Session timeout dài (30 ngày).
    Tiết lộ thông tin lỗi chi tiết (DB error: ... Query: ...).
    Phân biệt user không tồn tại / sai mật khẩu.
    Không giới hạn số lần đăng nhập, không captcha.
    Cho phép đăng nhập nhiều thiết bị cùng lúc.
- register.php:
    SQL Injection trực tiếp khi INSERT ('$username', '$password' không escape).
    Lưu mật khẩu dạng plaintext.
    Không kiểm tra độ dài input → DoS.
    Không kiểm tra username tồn tại trước khi insert → lỗi SQL.
    Tiết lộ lỗi MySQL & câu query ra giao diện.
    Không xác thực định dạng input → có thể chứa HTML/JS → Stored XSS.
    Không giới hạn số lần đăng ký → spam account.
    ID tuần tự, dễ đoán.
    Session fixation (cho phép set sid qua GET, không regenerate).
    Cookie chứa plaintext credential.
    Second Order SQL Injection: Dữ liệu từ $username được lưu vào bảng profiles mà không escape, có thể nổ ở chỗ khác khi truy xuất.
- products.php
    SQL Injection: mọi tham số lọc/sort/search được ghép chuỗi vào SQL (primary SQLi).
    Reflected XSS: in thẳng tham số q (search) vào HTML.
    Stored XSS: hiển thị name/description từ DB mà không escape.
    IDOR / Insecure object reference: product detail dùng id từ query string và hiển thị dữ liệu mà không kiểm tra quyền (public page, nhưng id dễ đoán).
    Information disclosure: hiển thị lỗi SQL chi tiết nếu query fail.
    CSRF: form Add to cart POST không có CSRF token.
    Open redirect: tham số return được dùng để redirect sau add-to-cart mà không validate.
    Unvalidated file path: ảnh sản phẩm dùng đường dẫn từ DB trực tiếp (có thể dẫn tới path traversal nếu upload/DB bị tấn công).
    DoS / performance: không giới hạn size input, không throttle, query không giới hạn kết quả nếu không có pagination.
    Second-order SQLi: nếu product fields chứa payload, sau này sử dụng trực tiếp sẽ nổ (để demo kết hợp với register/profile).
    Reflected param-based sorting injection: sort và order ghép thẳng vào SQL.
